# ТІЛДЕС — Платформа бесплатного обучения казахскому языку

> *Тілдес — "языковой товарищ" / "единомышленник по языку"*

---

## Общая концепция

Веб-платформа для бесплатного изучения казахского языка с живыми преподавателями, групповым форматом обучения, социальной лентой и полноценной админ-панелью.

**Финансирование** — государственный социальный бюджет.  
**Преподаватели** — работают по принципу фриланс-биржи, получают оплату из общего государственного фонда за каждый проведённый урок.

---

## Роли на платформе

| Роль | Описание |
|---|---|
| **Пользователь → Ученик** | Регистрируется, проходит верификацию и обучение |
| **Преподаватель** | Верифицированный специалист, проводит уроки |
| **Администратор** | Управляет платформой через админ-панель |

---

## ВКЛАДКА 1 — Обучение (основная)

### Путь ученика

#### Шаг 1 — Верификация личности
- Вводит ИИН и данные удостоверения личности
- Данные уходят на модерацию
- После одобрения присваивается статус **"Ученик"** и начальный ранг

**Ранговая система:**
```
Бастауыш → Орта → Жетік → Шебер → Ұстаз
```

---

#### Шаг 2 — Вступительный тест (25 вопросов)
- Вопросы с вариантами ответа
- 2–3 голосовых вопроса — нужно ответить **текстом** (проверка восприятия на слух)
- 1 задание — **записать голосом** чтение текста на казахском
- Все данные обрабатываются LLM, выдаётся уровень владения языком

**Уровни:** `A1` / `A2` / `B1` / `B2` / `C1`

---

#### Шаг 3 — Выбор группы (Поток)
- Размер группы: **от 2 до 5 человек**
- Показываются только группы похожего уровня (±1 ступень)
- Доступные расписания:
  - 1 раз в неделю
  - 3 раза в неделю
  - Интенсив
- Можно **создать свою группу** и пригласить друзей по ссылке

---

#### Шаг 4 — Обучение в группе

Каждая группа имеет:

- 📋 **План обучения** — разбитый на секции и этапы
- 🗺️ **Карта прогресса** — визуальное отображение пройденного и предстоящего
- 💬 **Чат группы** — общение, материалы, домашние задания
- 📚 **ДЗ** — выдаётся автоматически после урока по текущему плану

---

### Повышение уровня — Экзамен

Чтобы получить уровень выше, ученик сдаёт **экзамен**:
- 20 вопросов с вариантами ответа
- Голосовое задание — произнести предложение, LLM обрабатывает и выносит вердикт
- По результату присваивается следующий уровень

**Когда можно сдавать:**
- **Автоматически** — по завершении потока/группы
- **Досрочно** — раз в две недели по желанию ученика

## ПРЕПОДАВАТЕЛЬ

### Регистрация и верификация
1. При регистрации выбирает роль **"Преподаватель"**
2. Заполняет анкету-резюме: образование, опыт, уровень казахского
3. Заявка уходит на модерацию
4. Центр связывается с кандидатом, проводит **онлайн-созвон**: проверка уровня языка и педагогических навыков
5. После одобрения — полный доступ к аккаунту преподавателя

### Работа на платформе
- Видит список **групп без преподавателя** с указанием уровня, текущей секции и этапа по плану
- Выбирает группу, назначает дату и время урока
- Может **запланировать уроки на несколько недель вперёд** для одной группы
- Может отменить урок *(не более N раз в месяц — иначе снижается приоритет в бирже)*

### Проведение урока
- За **6 часов до урока** открывается доступ к чату группы
- Может заранее скинуть материалы, поздороваться, подготовить учеников
- Платформа **автоматически создаёт Zoom-конференцию** и публикует ссылку в чат
- Урок длится **1 час**
- После урока платформа **автоматически публикует ДЗ** в чат по текущему плану

### Проверка ДЗ
- ДЗ проверяет **следующий преподаватель**, который берёт эту группу
- Это мотивирует изучать контекст группы перед уроком

### Оценка и оплата
- После каждого урока ученики ставят оценку **от 1 до 5**
- Можно оставить "анонимный" отзыв *(в админ-панели видно, кто оставил — но тихо)*
- Оплата из **государственного фонда** за каждый проведённый урок
- Чем выше рейтинг преподавателя — тем выше ставка

## ВКЛАДКА 2 — Лента (Threads-формат)

Только на **казахском языке**. Посты на других языках не проходят модерацию.

### Функционал постов
| Действие | Описание |
|---|---|
| ❤️ Лайк | Оценить пост |
| 🔁 Репост | Поделиться постом |
| 💬 Комментарий | Оставить комментарий |
| 🚩 Жалоба | Пожаловаться с выбором причины |

### Профиль пользователя
- Аватар, имя, ранг
- Посты и репосты пользователя
- Подписчики / подписки
- Общее количество лайков

## АДМИН-ПАНЕЛЬ

### Дашборд (главная страница)
- Всего пользователей на платформе
- Всего учеников
- Всего преподавателей
- То же самое — но **за последние сутки**

---

### Вкладка — Модерация

**Подвкладка: Пользователи**
- Список жалоб на аккаунты
- Причина жалобы, дата, автор жалобы
- Действия: предупреждение / блокировка аккаунта

**Подвкладка: Посты**
- Список жалоб на посты
- Причина жалобы, дата
- Действия: удалить пост / предупредить автора

---

### Вкладка — Посты
- Общее количество постов
- Количество постов за сутки
- Средняя активность пользователей *(заглушка с реалистичными данными)*

---

### Вкладка — Преподаватели
- Список всех преподавателей
- Поиск по имени
- Фильтрация:
  - По средней оценке (возрастание / убывание)
  - По дате добавления (возрастание / убывание)

**При клике на преподавателя:**
- Средняя оценка
- Все отзывы с указанием **кто оставил** *(видно только администратору)*
- Дата добавления на платформу

---

### Вкладка — Ученики

**Статистика:**
- Сколько учеников повысило уровень за сутки
- Столбчатый график — распределение учеников по уровням
- Кнопка переключения: **количество ↔ процент от всех учеников**

**Список учеников:**
- Поиск по имени
- Фильтрация блоками по уровню: `A1` / `A2` / `B1` / `B2` / `C1`
- Сортировка по дате регистрации (возрастание / убывание)
- Кнопка **БАН**

## Сравнение с конкурентами

| Критерий | Конкуренты | ТІЛДЕС |
|---|---|---|
| Формат обучения | Дуолинго-клоны, мини-игры | Живые уроки с преподавателем |
| Верификация пользователей | Нет | ИИН + удостоверение личности |
| Групповое обучение | Нет | Персонализированные группы по уровню |
| Повышение уровня | Очки и стрики | Реальный экзамен с обработкой LLM |
| Социальная составляющая | Нет | Threads-формат, только казахский |
| Финансирование | Несуществующие спонсоры | Государственный социальный бюджет |
| Админ-панель | Нет | Полноценная с аналитикой и модерацией |
