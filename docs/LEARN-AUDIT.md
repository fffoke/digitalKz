# Аудит вкладки «Обучение» — что не так и что улучшено

## Найденные проблемы
1. **Две секции «Уроки»** у ученика — одна (в блоке информации) без оценки, вторая (глобальная) с оценкой. Дублирование.
2. **Остаточный div с ДЗ** — встроенная секция «Домашние задания» рендерилась глобально по `overview.homeworks`, а список ДЗ ученика не зависит от членства в группе → после выхода из группы блок оставался на экране.
3. **Аудио для аудирования только по URL** — в админке нельзя было загрузить файл или записать голос, только вставить ссылку (неудобно, негде взять URL).

## Что сделано
1. **Одна секция «Уроки»** — у ученика теперь единый блок уроков (внутри группы) с подключением и **оценкой завершённого урока**. Глобальная секция уроков оставлена только преподавателю («Мои уроки» — вести/завершать). Дубля нет.
2. **Убран остаточный ДЗ-блок** — встроенная секция ДЗ удалена; домашние задания живут на отдельной странице `/learn/homework` (кнопка «📒 ДЗ»). После выхода из группы лишних блоков не остаётся (секции уроков/прогресса — внутри `currentGroup`).
3. **Аудио загрузкой/записью** — в редакторе экзамена для вопроса «Аудирование»:
   - **📎 Загрузить файл** (audio/*),
   - **● Записать голос** (MediaRecorder),
   - предпросмотр плеером.
   Бэкенд: `POST /api/admin/exams/upload-audio` сохраняет файл в `/uploads/exam-audio` и возвращает URL, который подставляется в вопрос.
4. **Оценка урока — один раз** (ранее): повторная попытка → 409.

## Архитектура осталась прежней
Слои Repository → Service → Router; ДЗ и чат — отдельные страницы; админка на React.
